# Markdown 指南
Markdown 是一种轻量级的易用的书写语法。本文是 Markdown 的一个快速指南[^1]。

## 什么是 Markdown？
Markdown 是一种通过少量简单的标记字符来格式化文本的方法。您可以用纯文本来书写文档，然后在阅读时呈现一个美观的排版。

其实并没有一个标准的 Markdown 语法，很多编辑器都会添加自己的扩展语法。不同于此，为了兼容性，VNote 仅仅支持那些被广泛使用的基本语法。
