# PXF 手册
Greenplum Platform Extension Framework (PXF)通过将Greenplum数据库外部表定义映射到外部数据源的内置连接器，提供了跨异构数据源的并行、高吞吐量数据访问和联邦查询。PXF起源于Apache HAWQ项目。
