# bash-suport插件介绍
 bash-suport 长江为vim 实现了一个ide. 位的是加快bash 代码编写.通过插入完整的语句,注释,习惯用法和代码片段实现.所有的语法检查,运行脚本,启动调试器都可以通过快捷键暗沉.可以提高编写bash 脚本的舒适度.
