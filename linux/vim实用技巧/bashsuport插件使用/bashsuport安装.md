# Vundle 方式安装
1. [安装Vundle](/zh/linux/vim实用技巧/Vundle插件管理器安装).
2. 添加以下代码到`.vimrc`文件中
```vim

"""""""""""""""""" shell 编写IDE""""""""""""""""""""""
Plugin 'WolfgangMehner/bash-support'
```
3. 打开`vim` ,执行`PluginIstall` 
4. 安装完毕
